# 同花顺策略公式使用说明

1. 在桌面程序的“同花顺策略”页完成回测。
2. 双击结果表中的策略，或右键选择“策略卡片 / 导出同花顺公式”。
3. 策略卡片提供四种公式：
   - 买入选股公式：用于条件选股，只输出当前买入条件。
   - 同花顺回测公式：用同一组 `BUYSIG` / `SELLSIG` 输出 `DRAWICON` 和 `DRAWTEXT`，供同花顺回测窗口识别“买入”“卖出”点。
   - 交易系统公式：用 `ENTERLONG` 和 `EXITLONG` 在交易系统中输出买卖信号。
   - K线标注公式：与回测公式共用完全相同的买卖信号，在 K 线上标出买卖点。
4. 在同花顺桌面端打开自定义公式编辑器，新建对应类型的公式，粘贴后先执行语法测试。
5. 公式默认按日线收盘数据计算。实盘使用前，应核对复权方式、周期、历史箭头和下一交易日可成交性。

同花顺“回测”窗口应粘贴“同花顺回测公式”，不要粘贴仅含 `XG:BUYCOND;` 的买入选股公式。回测公式仅在条件从不成立变为成立时标出一次买卖点，图标和“买入/卖出”文字由同一个信号驱动。

策略卡片顶部的“买入公式”和“卖出公式”直接展示 `BUYCOND:=...;`、`SELLCOND:=...;` 可执行表达式。诸如“量比适中”“资金转弱”等自然语言仅作为交易解读，不会再显示成可复制的交易条件。

## 数据兼容性

- MACD、KDJ、RSI、BOLL、WR、CCI、BIAS、MTM、均线、成交量和成交额策略可生成普通公式。
- 本地程序使用真实资金流的策略，在导出时会明确标注为“OBV 技术代理版”，其信号不会与本地回测完全一致。
- 龙虎榜和机构席位策略依赖本地收盘后事件数据，普通公式编辑器无法获得同一数据源，因此不会生成虚假的等价公式。
- “获利盘超过 90%”在同花顺公式中使用 `WINNER(CLOSE)>0.90`。本地回测只有在数据包含筹码获利比例字段时才启用该条件。

同花顺不同券商版本的公式编辑器可能存在函数差异。若某个函数无法编译，应保留错误提示并核对该客户端的公式说明书，不要在未核对的情况下用于实盘。
